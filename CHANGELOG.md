# Changelog

All notable changes to XLSX Viewer are documented here.

## 0.1.0 - 2026-06-07

- Initial local release.
- Add read-only `.xlsx` workbook view.
- Add sheet tabs, sticky headers, row numbers, search/filter, and workbook summary.
- Add empty sheet, malformed workbook, and large-file render-cap states.
- Show cached/formatted cell values without evaluating formulas or writing files.
